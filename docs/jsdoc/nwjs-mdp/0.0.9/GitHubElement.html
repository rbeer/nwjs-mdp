GitHubElement

GitHubElement

GitHub API rate limit status component

Constructor

new GitHubElement(rootElementopt)

Creates the widget's elements inside a given root
The root element can be placed and styled at will, GitHubElement will not modify/add/remove any styles/classes on it.

Parameters:
Name Type Attributes Default Description
rootElement HTMLDivElement <optional>
 document.querySelector('[data-github-element]')

container for component
Source:
  • lib/github-element.cls.js, line 34
Throws:

If passed rootElement is not an instance of HTMLDivElement and no element with 'data-github-element' attribute was found.

Type
TypeError
Examples

Setup HTML:

<body>
  <statusbar>
    <div class="pull-right" data-github-element></div>
  </statusbar>
</body>

Setup JavaScript:

let rootDiv = document.querySelector('statusbar div');
let githubElement = new GitHubElement(rootDiv);
// ... new GitHubElement(); works too, because the <div>
// has the data-github-element attribute.

Result HTML:

<body>
  <statusbar>
    <div class="pull-right" data-github-element>
      <i class="fa fa-pull-right fa-github"></i>
      <div class="pull-right title-text github-ratelimit hidden">
        <span data-github-ratelimit-limit>--</span> / <span data-github-ratelimit-remaining>--</span>
      </div>
    </div>
  </statusbar>
</body>
